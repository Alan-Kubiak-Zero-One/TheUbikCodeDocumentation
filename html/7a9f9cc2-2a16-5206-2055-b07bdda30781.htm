ScriptManagerRegisterHostFunction Method

Overload List
  NameDescription
RegisterHostFunction(HostFunctionPrototype)
Registers the given HostFunctionPrototype without a handler. If a Script uses the given host function, the handler must be bound at ScriptContext level.
RegisterHostFunction(HostFunctionPrototype, HostFunctionHandler)
Registers the given HostFunctionPrototype with an accompanying HostFunctionHandler. Handlers defined at ScriptContext level for this function are ignored.
