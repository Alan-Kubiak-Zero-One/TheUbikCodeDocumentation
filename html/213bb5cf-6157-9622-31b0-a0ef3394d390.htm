ScriptContext Properties

The ScriptContext type exposes the following members.

Properties
  NameDescription
ChildThreads
Child thread contexts generated by the script.
FunctionStack
Current execution function stack.
Handler
Context-level host function handler. This handler is ignored by host functions with handlers defined at ScriptManager level.
Interrupted
Boolean flag indicating if an interrupt was generated after the last execution run. The flag is cleared in subsequent executions.
InterruptOnHostfunctionCall
Boolean flag to enable or disable interrupts whenever a host function is invoked.
LocalDictionary
Variable dictionary defined at local scope.
NextInstruction
Index to the next instruction to be executed.
ParameterStack
Current parameter value stack.
Script
Script associated with this context.
Terminated
Boolean flag indicating if the initially specified function has completed execution or otherwise.
